# Simple Blog API

A simple backend API for managing users, posts, and comments built with **Node.js** and **Express.js**.  
This project demonstrates RESTful API design, middleware usage, error handling, and in-memory data storage.

## Features

- **User Management**
  - Register/Login users
  - Get user profile
  - Update user information

- **Post Management**
  - Create, read, update, delete posts
  - Get posts by user
  - Search posts by title/content

- **Comment System**
  - Add comments to posts
  - Get comments for a post
  - Delete comments

- **Middleware**
  - Logging requests
  - Basic authentication
  - Error handling (404 & 500)
